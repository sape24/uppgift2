Om sidan

Webbplatsen är en enkel CRUD-applikation som hanterar arbetserfarenheter där användaren kan se en lista över tidigare arbetserfarenheter, lägga till nya via ett formulär samt att radera befintliga.


Webbplatsen byggdes med hjälp av Node.js och express till backend med sqlite3 som databas, frontend är byggd med ren HTML, CSS och javascript samt parcel


Jag skulle vilja påstå att jag lärt mig hur man skapar en REST-API, hur man använder CORS och användning av det samt blivit bättre med asynkrona funktioner även om vi har arbetat med det tidigare.
